 
Задание 3: 
Используя Visual Studio, создайте проект по шаблону ASP.NET Empty Web Application. 
Создайте обработчики для события Load и Unload.  
Добавьте в проект папку App_Data и поместите нее текстовый файл с производным текстом. 
При обработке события Load создайте объект StreamReader и с его помощью считайте данные из файла и отобразите в теле страницы. 
При обработке события Unload вызовите метод Close на объекте класса StreamReader для того, чтобы освободить системные ресурсы, которые выделились при работе с файлом.
